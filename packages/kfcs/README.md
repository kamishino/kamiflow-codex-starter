# KFCS

Web-first Codex session manager utility.

## Purpose

KFCS is separate from `kfc`.

- `kfc`: Kami Flow workflow and client-project automation
- `kfcs`: Codex session browsing, import/export, and restore help

## Commands

```bash
kfcs serve
kfcs index
kfcs where
kfcs find --id <session-id>
kfcs export --id <session-id> --to <path>
kfcs import --from <path>
kfcs restore --id <session-id>
```
