# KFC Session

Web-first Codex session manager utility.

## Purpose

KFC Session is separate from `kfc`.

- `kfc`: Kami Flow workflow and client-project automation
- `kfc-session`: Codex session browsing, import/export, and restore help

## Commands

```bash
kfc-session serve
kfc-session index
kfc-session where
kfc-session find --id <session-id>
kfc-session export --id <session-id> --to <path>
kfc-session import --from <path>
kfc-session restore --id <session-id>
```
