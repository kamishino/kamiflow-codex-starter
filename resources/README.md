# Resources Scaffold

Generic, reusable container for future Kami Flow Codex assets.

## Directories

- `skills/`: future Codex skills (`SKILL.md`-based).
- `prompts/`: future explicit prompt command files.
- `scripts/`: future helper scripts (JS/TS planned).
- `templates/`: reusable templates and assets.
- `docs/`: guidance and rollout docs for this resource pack.

This folder intentionally contains only placeholders right now.
